 
# 📌 Responsive Blog Layout (Tailwind CSS)

A modern and responsive blog layout built using **HTML, Tailwind CSS, and JavaScript**.
This project features a clean UI with a sidebar, blog cards, and interactive modal functionality.

---

## 🚀 Features

* 📱 Fully Responsive Design
* 🎨 Styled using Tailwind CSS
* 📂 Sidebar Navigation (Toggle Menu)
* 📰 Blog Cards Layout
* 🔍 "Read More" Modal Popup
* ✨ Smooth Animations & Hover Effects
